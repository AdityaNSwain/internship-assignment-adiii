What is Git?
Git is a version control system used to track changes in files and code.

Why Git is useful?
Git helps developers save versions, undo mistakes, work in teams, and manage project changes easily.

Difference between Git and GitHub:
Git is a tool used on your computer to track code changes.
GitHub is an online platform where Git projects are stored and shared.

Three commands learned:
mkdir – creates a folder
cd – changes directory
ls – lists files and folders
